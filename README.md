# MealHut Sri Lanka
+ This e-commerce web app for restaurants features user-friendly interfaces, admin/customer logins, product catalog, shopping cart, checkout, payment gateway, order history, and tracking.
## Technologies
+ Java EE `javaee`
+ Hibernate `hibernate`
+ HTML `html`
+ CSS `css`
+ JavaScript `js`
+ Bootstrap `bootstrap`
+ MySQL `mysql`
